# Unique Pools & General Maintenance - Website Content Plan

## 1. Goal
Attract new residential and commercial clients for pool, spa, and sauna maintenance and construction. 
**SEO Strategy:** Focus on local keywords (e.g., "pool maintenance Karachi", "swimming pool construction Pakistan"), high-quality service descriptions, and clear call-to-actions (CTAs).

## 2. Site Architecture
- **Home:** Hero section, summary of services, key value prop (25+ years exp), CTA.
- **Services:** Detailed breakdown of pools (civil/MEP), spas/sauna, waterfalls/fountains, water parks.
- **Projects (Portfolio):** Visual showcase of key work (Mövenpick, resorts, etc.).
- **About Us:** Established 1989, national/international expertise.
- **Contact:** Form, phone, email, location.

## 3. SEO Compliance
- **Meta tags:** Unique title/description for each page.
- **Semantic HTML:** Use `<h1>` to `<h3>` tags correctly.
- **Alt text:** Descriptive text for all imagery.
- **Performance:** Clean, minimal code (Stripe-inspired system) for fast loading.
